## 📋 ファイル別ガイドライン

| ファイル| 説明 | コミット可？ | 理由 |
|---------|------|-----------|------|
| `package.json` | 依存パッケージ定義 | ✅ はい | 公開情報 |
| `tsconfig.json` | TypeScript 設定 | ✅ はい | 公開情報 |
| `.env` | 環境変数 | ❌ いいえ | API キー含む |
| `credentials.json` | 認証ファイル | ❌ いいえ | 秘密鍵含む |
| `src/` | ソースコード | ✅ はい | 公開情報 |
| `node_modules/` | 依存パッケージ | ❌ いいえ | package.json から復元可能 |
| `dist/` | ビルド出力 | ❌ いいえ | ソースから生成可能 |
| `.DS_Store` | macOS ファイル | ❌ いいえ | システムファイル |

## 🚨 万が一秘密情報をコミットしてしまった場合

### 1️⃣ 即座に対応
```bash
# 最新のコミットから削除
git rm --cached <秘密ファイル名>
git commit --amend --no-edit
git push -f origin main
```

### 2️⃣ 認証情報をリセット
- Google Cloud: 該当サービスアカウントを削除
- API キー: 新しいキーを生成して古いキーを無効化
- SSH キー: 新しい鍵を生成

### 3️⃣ GitHub Secret Scanning をチェック
```
https://github.com/yakinorinori/CompanyWebsite/security/secret-scanning
```
